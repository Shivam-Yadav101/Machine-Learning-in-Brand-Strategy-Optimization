# Machine-Learning-In-Brand-Mastering-Strategies
Machine Learning project that predicts retail sales and optimizes pricing strategies using XGBoost to improve company efficiency and maximize revenue.

#  Machine Learning in Brand Strategy Optimization

## Project Overview
This project focuses on using Machine Learning to optimize brand strategies and improve company efficiency in the Retail 4.0 ecosystem.

It predicts sales volume and provides data-driven insights for:
- Dynamic pricing
- Discount optimization
- Demand forecasting

---

## Objective
- Predict sales using ML models
- Compare Linear Regression, Random Forest, and XGBoost
- Build a Strategy Optimization Engine to find optimal discount range

---

## Models Used
- Linear Regression (Baseline)
- Random Forest Regressor
- XGBoost Regressor (Best Model)

---

## Key Results
- XGBoost achieved highest accuracy
- R² Score ≈ 0.89
- RMSE significantly reduced compared to baseline

---

## Business Insights
- Optimal discount range: **15% – 25%**
- Avoid over-discounting (reduces brand value)
- Store size and region strongly impact sales

---

## Dataset Features
- Price_MRP
- Discount_Percent
- Store_Size
- Region
- Outlet_Type
- Competitor Presence
- Seasonal Trends
